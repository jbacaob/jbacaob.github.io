# jbacaob.github.io

Source for [jbacaob.com](https://www.jbacaob.com/), the personal academic website of Jafet Baca, a Ph.D. Candidate in Economics at Emory University. The site hosts his research, teaching materials, CV, and blog.

Built with [Jekyll](https://jekyllrb.com/) using the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy/) theme.

## Local development

```console
$ bundle
$ bundle exec jekyll serve
```
